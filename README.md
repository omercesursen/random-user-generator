# random-user-generator

Öncelikle yeni bir proje klasörü açtım ve virtual environment (venv) oluşturdum.

Api'ye istek atabilmek için requests kütüphanesini pip install komutu ile kurdum.
Requests kütüphanesinin nasıl kullandığına baktım.
Siteden önce tek veri çekip terminalde yazdırmayı denedim.
Başarılı olunca birden fazla kullanıcı çekmek için kullanılan parametreyi araştırdım. Sitesinde bu parametrenin results olduğunu öğrendim ve koduma ekledim.
Son olarak Pythonda nasıl dosyaya yazı yazdırılır diye araştırdım ve kodumda uyguladım.

Sonuç olarak kullanıcıdan alınan sayı kadar veriyi çekip, .txt dosyasına yazdıran bir uygulama hazırladım.
